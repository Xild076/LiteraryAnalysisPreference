# Data Schema

This document will outline the procedure for annotating this data.

The data will be stored in a **CSV file** with the following structure:

```csv
poem_text, poem_fetch_url, poem_genre, author_name, author_age, author_gender, author_ethnicity, author_nationality
...
```

**Poem text:** The full text of the poem
**Poem fetch URL:** The link from which the full text of the poem can be found
**Poem Genre:** The genre of the poem
**Author Name:** Full name of the author
**Author Age:** Age of the author at the time of writing the poem
**Author Gender:** The gender of the author 
**Author Ethnicity:** The ethnicity of the author
**Author Nationality:** The nationality of the author

For each fo the author specifics, it may be labeled as "anonymous"
